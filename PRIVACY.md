# CatchIt 隐私说明

最后更新：2026-07-14

CatchIt 是本地优先的 macOS 截图工具。

## 数据处理

- 截图、标注和便签仅在你的 Mac 上处理。
- CatchIt 不上传截图，不收集使用分析、设备标识、联系人或账号信息。
- 截图默认保存在 `~/Pictures/CatchIt/`，也可以由用户选择其他目录。
- “检查更新”只在用户主动点击时访问 GitHub Releases，并由 GitHub 按其隐私政策处理网络请求。

## 系统权限

- 屏幕录制：用于截取用户选择的屏幕或区域。
- 登录时启动：仅在用户主动开启后注册 CatchIt 登录项。

## 数据保留与删除

用户可以永久保留截图，或在“存储管理”中选择保留期限并把过期截图移到废纸篓。卸载 CatchIt 不会自动删除截图目录。

## 开源代码

CatchIt 的源码与版本发布计划通过项目 GitHub 仓库公开。正式仓库地址会在首个公开版本发布时补充。
